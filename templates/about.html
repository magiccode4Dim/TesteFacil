Teste Fácil

O Teste Fácil é uma plataforma de código feixado desenvolvida pela Magic Code , para facilitar o processo de Avaliação de Estudante A Plataforma esta equipada com recursos que permitem a correção Automatica de Provas Escolares bem como a disponibilização estantanêa de Resultados. O Teste Fácil também permite que se possa ver o desempenho academico dos estudantes, permitindo assim que o Professor ou Estudante passa ter uma visão geral do que precisa ser melhorado para que se possa ter resultados Academicos mais satisfatórios.

A Plataforma foi pensada para se adequar a realidade da maioria das Instituições em Moçambique e pelo Mundo a fora. Ela não só pode ser usada para provas escolares, pode ser usada também em mini-testes e qualquer tipo de questionário.

O Teste Fácil tem suporte da equipa da Magic Code que trabalha para melhorar ainda mais a esperiência dos utilizadores nesta plataforma . Actualmente, a nossa equipa trabalha no desenvolvimento de modulos de Inteligência Artifical que irão possibilitar a correção automatica de perguntas abertas e analise de dados mais detalhadas

A Magic Code é uma empresa Moçambicana de Desenvolvimento de Software. A nossa equipa trabalha com softwares por encomenda como blogs, sistemas mobile e Desktop. Trabalhamos também com API's.
